# 超级导师前端开发需求 V1.0

## 1. 技术栈
- PySide6 (Qt for Python)
- 无额外 UI 框架

## 2. 整体布局

```
┌─ 标题栏（可拖拽 + 窗口控制）──────────────────────────────┐
│ 🧠 超级导师 Super-Tutor  [课程 ▼] [+][✎][×]      [⚙️]   │
├───┬──────────────────────────────────────────────────────┤
│ 左 │  中 央 区 域                                         │
│ 侧 │  [💬 问答]  [📅 规划+进度]                          │
│ 面 │                                                      │
│ 板 │                                                      │
├───┴──────────────────────────────────────────────────────┤
│ ● 状态信息                                               │
└──────────────────────────────────────────────────────────┘
```

### 2.1 窗口
- 无边框窗口（Qt.FramelessWindowHint）
- 最小 1100x700，默认 1300x850
- 圆角 14px，带阴影
- 可拖拽标题栏

### 2.2 标题栏（高 52px）
| 元素 | 说明 |
|------|------|
| 🧠 图标 + "超级导师" 标题 | 左对齐 |
| 课程下拉框 | 显示当前课程，点击展开列表 |
| [+][✎][×] | 新建/重命名/删除课程 |
| [⚙️] | 打开设置对话框 |
| [—][□][✕] | 最小化/最大化/关闭 |

### 2.3 颜色主题
- 背景 #0f1117（极深灰）
- 表面 #1c2030（卡片）
- 边框 #2a2e3d
- accent #6c63ff（紫蓝）
- 主文字 #e8eaf0
- 次要文字 #8b8fa3
- 成功 #34d399（绿）
- 错误 #ef4444（红）

## 3. 左侧面板（宽 280px，可调）

### 3.1 知识库文档树
- 按课程分组显示文档
- 分组节点：「📖 教材」「📝 真题」
- 文档项显示文件名
- 右键菜单：重命名 / 删除
- 底部：「📄 上传文档」按钮（accent 色）

### 3.2 空状态
- 无文档时显示「📚 暂无文档」

## 4. 问答 Tab

### 4.1 布局
```
┌─ 💡 输入问题开始学习（状态标签）──────────────────────────┐
├─ 聊天记录（QTextBrowser，深色背景，圆角 10px）────────────┤
│                                                          │
│  用户：xxx                                              │
│  助手：xxx  [来源: 文档]                                 │
│                                                          │
├─ [📝 做本章测验] ────────────────────────────────────────┤
├─ [💬 输入问题...                    ] [📤发送] [■停止]   │
└──────────────────────────────────────────────────────────┘
```

### 4.2 聊天气泡
- 用户消息：左侧紫色边框，深色背景
- 助手消息：正常文本，末尾显示来源标注
- 支持 Markdown 渲染

### 4.3 输入区
- 圆角输入框，聚焦显示 accent 边框
- 发送按钮 accent 色
- 停止按钮红色

### 4.4 流式渲染
- `token_signal` 逐 token 追加到 `QTextBrowser`
- 自动滚动到底部

## 5. 规划+进度 Tab

### 5.1 参数设置
```
目标: [30 ▼] 天  每天: [2 ▼] 小时  [📅 生成计划]
```

### 5.2 每日任务卡片
```
📅 第 1 天 — 第一章 绪论
  ☑️ 1.1 数据结构概念
  ☑️ 1.2 算法分析
  ⬜ 1.3 数学基础
  ─────────────────────
  已完成 2/3  ████████░░
```

- 每张卡片深色背景，圆角 10px
- 复选框自定义样式（accent 选中色）
- ✅ 打勾后进度条即时刷新

### 5.3 整体进度
```
████████░░░░░░░░░░░░ 30% (9/30 天)
已掌握章节：第1章(100%) ✅  第7章(25%) ⚠️
```

### 5.4 操作按钮
| 按钮 | 行为 |
|------|------|
| 📝 做题 | 对当前章节触发 AI 出题测验 |
| 🔄 重新生成 | 跳过已学章节，重新生成计划 |
| 📤 导出计划 | 导出 Markdown |

## 6. 设置对话框

### 6.1 字段
| 字段 | 默认值 |
|------|--------|
| API Key | （密码框） |
| API Base | https://api.deepseek.com/v1 |
| 模型 | deepseek-chat |
| 存储目录 | knowledge_base |

### 6.2 行为
- 保存到 `.env` 文件
- 保存后弹窗提示「配置已保存，重启后生效」

## 7. 课程管理

### 7.1 数据格式（courses.json）
```json
[{
  "id": "uuid",
  "name": "课程 1",
  "organize_by": "auto",
  "created_at": "2026-07-01T12:00:00"
}]
```

### 7.2 操作
- 新建：默认名「课程 N」
- 重命名：QInputDialog
- 删除：确认弹窗，清除关联数据

## 8. 状态栏
- 左侧：● 绿色圆点 + 状态文本
- 深色背景，36px 高

## 9. 验收标准

| # | 验收项 | 操作 |
|---|--------|------|
| 1 | 启动后显示无边框深色窗口 | python main.py |
| 2 | 标题栏可拖拽 | 拖拽标题栏移动窗口 |
| 3 | 课程新建/改名/删除 | 操作课程下拉框 |
| 4 | 上传文档到知识库树 | 点击上传按钮选文件 |
| 5 | 问答界面多轮对话 | 输入问题→发送→显示 |
| 6 | 生成计划显示卡片+进度条 | 参数→生成计划 |
| 7 | 打勾后进度刷新 | 勾选复选框 |
| 8 | 设置弹窗保存 | 填入→保存→检查 .env |
| 9 | 状态栏显示当前课程名 | 切换课程观察变化 |

## 10. 设计红线（Impeccable）

- ❌ 不要用奶油色/米色背景
- ❌ 不要用灰色文字在彩色背景上
- ❌ 不要卡片嵌套卡片
- ❌ 不要侧边条纹装饰
- ❌ 不要渐变文字
- ✅ 使用 OKLCH 色彩
- ✅ 单字体族（Segoe UI / PingFang SC / Microsoft YaHei）
- ✅ 圆角 8-14px，不过度
